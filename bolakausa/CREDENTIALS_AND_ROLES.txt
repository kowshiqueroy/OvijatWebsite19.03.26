3. ROLE PERMISSION & ACCESS DIRECTORY
--------------------------------------------------------------------------------

A. STAFF ROLES (BACKEND SYSTEM)
--------------------------------
1. ADMIN (Role Rank: 7)
   - Access URL: http://localhost/bolakausa/admin
   - Permission Scope: Full access to the entire application.
   - Core Features:
     * View executive financial metrics & system logs.
     * Approve bank deposits and Stripe top-up requests.
     * Edit site configuration, timezones, and pricing matrix values.
     * Review system audits, email queues, and transaction histories.
     * Oversee inventory logs, chats, orders, and products.

2. MANAGER (Role Rank: 6)
   - Access URL: http://localhost/bolakausa/manager
   - Permission Scope: Manage operations and inventory intake.
   - Core Features:
     * Access the Manager Dashboard.
     * Log new stock batches (Stock In) with shelf locations and lot numbers.
     * Capture webcam/selfie proof images of inventory lots.
     * Adjust damaged, expired, or returned lot levels.
     * Verify and approve pending users whose roles are below Manager.

3. EDITOR (Role Rank: 5)
   - Access URL: http://localhost/bolakausa/admin
   - Permission Scope: Manage product catalog and orders.
   - Core Features:
     * Access the product list and add/modify categories and catalog listings.
     * View, edit, or cancel customer orders.
     * Verify and approve pending users whose roles are below Editor.
     * Blocked from financial controls (wallet approvals, global settings, audit logs).

4. VIEWER / AUDITOR (Role Rank: 4)
   - Access URL: http://localhost/bolakausa/viewer
   - Permission Scope: Read-only financial overview and verification audits.
   - Core Features:
     * Access the Auditor Dashboard.
     * View system user list and account directories.
     * Blocked from all write/modify actions, settings, or catalog editing.

5. WAREHOUSE (Role Rank: 3)
   - Access URL: http://localhost/bolakausa/warehouse
   - Permission Scope: Order packing and logistics fulfillment.
   - Core Features:
     * Access the active fulfillment queue (Pending/Processing orders).
     * Generate packing checklists with FIFO (First-In, First-Out) lot matches.
     * Locate inventory by matching assigned shelf zones (e.g. A-01, B-02).
     * Blocked from all other staff pages.


B. CUSTOMER ROLES (FRONTEND STOREFRONT)
----------------------------------------
6. EXECUTIVE (Role Rank: 2)
   - Access URL: http://localhost/bolakausa/
   - Permission Scope: VIP Wholesale Client.
   - Core Features:
     * Browse VIP products with custom executive discount rates.
     * Claim executive-only coupons (e.g., VIP15).
     * Enjoy a 10% automatic global discount at checkout.
     * Fund the internal wallet balance or pay using COD, Pay Later, or Stripe.

7. WHOLESALE USER (Role Rank: 1)
   - Access URL: http://localhost/bolakausa/
   - Permission Scope: Standard B2B Client.
   - Core Features:
     * Browse the catalog and purchase items at standard wholesale price tiers.
     * Apply standard coupon codes (e.g., WELCOME20).
     * Checkout using COD, Bank Transfer (deposit proof), Stripe test card, or Wallet balance.
     * Manage account addresses and submit wallet top-up requests.
     * Initiate real-time help chats with staff members.
